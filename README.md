# Ft_iputils
A set of small useful utilities for Linux networking.


## Ping

Will use:
- ICMP Echo Request (type 8)
- ICMP Echo Reply (type 0)

Verify:
- Reachability of a host
- Round trip time of a packet (RTT)
- Packet loss %
